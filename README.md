# Fabric RTI

Refer here for a full description of this - https://medium.com/@rodcolledge/microsoft-fabric-realtime-intelligence-with-azure-functions-and-event-hubs-f010a705297f

Prerequisites
- Azure Funtion App
- Azure Event Hub Namespace & Event Hub
- VS Code Azure Functions Project

Replace the contents of the Requirements.txt file with the code in the repo and execute pip install -r requirements.txt
Edit the local.settings.json file with your Event Hub connection string and name
Replace the contents of the function_app.py file with the code in the repo.
